# Deployment Notes

## GitHub Pages deployment

This is a static codebase. Upload the files to the root of a GitHub repository and enable GitHub Pages from the `main` branch.

## What to show in the client walkthrough

1. Executive Command Center
2. Launch Mumbai Weekend Spike
3. Run Agent Swarm
4. Review the recommended transfer
5. Approve it
6. Show the mock SAP STO + WMS task
7. Show the audit trail
8. Export the ROI proof pack

## Customization points

- Change demo SKUs in `src/data.js`
- Change risk scoring in `src/agents.js`
- Change UI language and metrics in `src/app.js`
- Change visual design in `styles.css`
